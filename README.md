# Portfolio
Collection of my Presented Work

This repository contains a simple portfolio landing page that links out to your other projects (it does not host their content).

Files:
- `index.html` — HTML scaffold for the portfolio.
- `styles.css` — extracted styles used by the page.
- `script.js` — page behavior and the list of project links.

Quick customization:
- Open `script.js` and update the `projects` object with your project titles and URLs.
- Edit `index.html` to add/remove `.site-item` elements pointing to the `data-site` keys.

To preview locally, open `index.html` in your browser or serve the folder with any static server.
